# CloseGist 功能说明文档

## 项目概述

**项目名称**：CloseGist  
**项目描述**：一个轻量级的代码片段管理工具，用于创建、编辑和分享代码片段（Gist）  
**用户模式**：单用户系统（只需密码登录）

---

## 开发技术栈

### 后端技术
- **语言**：Go 1.25.5
- **Web框架**：Echo v4.15.0
- **数据库ORM**：GORM v1.31.1
- **数据库**：SQLite（默认），支持 MySQL、PostgreSQL
- **认证**：基于密码的认证系统
- **其他**：
  - Chroma 用于代码语法高亮
  - Bleve 用于搜索功能

### 前端技术
- **CSS框架**：Tailwind CSS v3
- **JavaScript**：原生 JavaScript
- **模板引擎**：Go HTML 模板
- **图标**：SVG 图标
- **二维码生成**：QRCode.js
- **CDN资源**：
  - Tailwind CSS: `https://cdn.tailwindcss.com`
  - QRCode.js: `https://cdn.bootcdn.net/ajax/libs/qrcodejs/1.0.0/qrcode.min.js`

---

## 现有UI设计参考

### 设计风格
- **整体风格**：简洁现代，采用Tailwind CSS默认设计系统
- **颜色方案**：
  - 主色：蓝色（`text-blue-600`）
  - 背景：浅灰色（`bg-gray-100`）
  - 卡片：白色（`bg-white`）
  - 成功提示：绿色（`bg-green-500`）
  - 错误提示：红色（`bg-red-500`）

### 布局结构
- **响应式布局**：使用Tailwind CSS响应式类
- **导航栏**：固定顶部，包含Logo、应用名称和功能按钮
- **主内容区**：居中布局，最大宽度为7xl
- **卡片式设计**：使用`bg-white`、`shadow-sm`等类创建卡片效果

### 组件设计
- **按钮**：
  - 文本按钮：默认链接样式
  - 图标按钮：圆形图标按钮，悬停效果
  - 表单按钮：标准Tailwind按钮样式
- **表单**：
  - 输入框：默认Tailwind样式，必填项有验证
  - 下拉框：标准HTML select元素
  - 文本域：支持多行输入
- **模态框**：
  - 二维码弹窗：全屏遮罩，居中卡片
  - 确认对话框：简单的确认提示
- **列表**：
  - Gist列表：表格形式，包含标题、可见性、时间和操作按钮
  - 版本列表：简洁的版本历史记录

### 交互设计
- **悬停效果**：按钮、链接等元素有悬停反馈
- **点击反馈**：按钮点击有视觉反馈
- **下拉菜单**：设置菜单的展开/收起
- **模态框**：点击遮罩关闭弹窗
- **复制功能**：点击复制按钮后有成功提示

### 图标使用
- **导航图标**：列表、加号、齿轮等SVG图标
- **操作图标**：文档、二维码、复制等SVG图标
- **Logo**：简单的SVG图标，蓝色背景，白色线条

## 页面功能说明

### 1. 登录页 (/login)

**功能描述**：用户登录页面

**页面元素**：
- 标题："登录"
- 密码输入框（单字段，无用户名）
- 登录按钮

**交互逻辑**：
- 输入密码后点击登录
- 密码正确则跳转到列表页
- 密码错误则显示错误提示

---

### 2. Gist列表页 (/all)

**功能描述**：显示所有Gist列表，支持搜索和排序

**页面元素**：
- 标题："全部 Gist"
- 搜索筛选区：
  - 搜索框（按标题搜索）
  - 排序下拉框（创建时间/更新时间）
  - 顺序下拉框（升序/降序）
  - 筛选按钮
- Gist列表：
  - 每行显示一个Gist：
    - 标题（可点击跳转到详情页）
    - 可见性标签（公开/私有）
    - 创建时间
    - 操作按钮：
      - 原始文件（跳转到原始文件页）
      - 二维码（弹出二维码）
      - 复制地址（复制链接到剪贴板）
- 分页：
  - 上一页/下一页链接

**交互逻辑**：
- 点击标题跳转到详情页
- 点击原始文件跳转到原始文件页
- 点击二维码弹出二维码弹窗
- 点击复制地址复制链接

---

### 3. 新建Gist页 (/new)

**功能描述**：创建新的Gist

**页面元素**：
- 标题："新建 Gist"
- 表单：
  - 标题输入框（必填，只能包含字母和数字）
  - 文件名输入框（可选，默认为标题.txt）
  - 内容文本框（必填，多行）
  - 可见性下拉框（公开/私有）
  - 保存按钮

**交互逻辑**：
- 填写表单后点击保存
- 验证标题格式
- 创建成功后跳转到列表页
- 显示成功/失败提示

---

### 4. Gist详情页 (/:title/detail)

**功能描述**：显示Gist的详细信息

**页面元素**：
- 顶部区域：
  - 标题
  - 可见性标签（公开/私有）
  - 创建时间
  - 操作按钮（已登录可见）：
    - 编辑
    - 删除（带确认对话框）
- 内容区域：
  - 文件名
  - 操作按钮：
    - 原始文件
    - 二维码
    - 复制内容
  - 代码块（语法高亮）
- 底部区域：
  - 查看修订历史链接
  - 返回列表链接

**交互逻辑**：
- 点击编辑跳转到编辑页
- 点击删除弹出确认对话框，确认后删除
- 点击原始文件跳转到原始文件页
- 点击二维码弹出二维码弹窗
- 点击复制内容复制代码到剪贴板

---

### 5. 编辑Gist页 (/:title/edit)

**功能描述**：编辑现有Gist

**页面元素**：
- 标题："编辑 Gist"
- 表单：
  - 标题输入框（预填充当前值）
  - 文件名输入框（预填充当前值）
  - 内容文本框（预填充当前值）
  - 可见性下拉框（预选当前值）
  - 保存按钮
  - 取消按钮

**交互逻辑**：
- 修改表单后点击保存
- 保存时创建修订历史
- 点击取消返回详情页

---

### 6. 修订历史页 (/:title/revisions)

**功能描述**：显示Gist的所有历史版本

**页面元素**：
- 标题："Gist标题 - 修订历史"
- 版本列表：
  - 每行显示一个版本：
    - 版本号（v1, v2, ...）
    - 创建时间
    - 查看链接
- 返回详情链接

**交互逻辑**：
- 点击查看跳转到历史版本详情页
- 点击返回详情跳转到详情页

---

### 7. 历史版本详情页 (/:title/revisions/:version)

**功能描述**：显示某个历史版本的详细信息

**页面元素**：
- 顶部区域：
  - 标题 + 版本号
  - 创建时间
  - 可见性标签（该版本的可见性）
- 内容区域：
  - 文件名
  - 代码块
- 返回修订列表链接

**交互逻辑**：
- 点击返回修订列表跳转到修订历史页

---

### 8. 修改密码页 (/settings/password)

**功能描述**：修改登录密码

**页面元素**：
- 标题："修改密码"
- 表单：
  - 原密码输入框
  - 新密码输入框
  - 确认新密码输入框
  - 保存按钮

**交互逻辑**：
- 填写表单后点击保存
- 验证原密码是否正确
- 验证两次新密码是否一致
- 显示成功/失败提示

---

### 9. 二维码弹窗

**功能描述**：显示Gist原始文件页的二维码

**页面元素**：
- 全屏遮罩
- 居中卡片：
  - 二维码图片
  - 对应链接

**交互逻辑**：
- 点击遮罩关闭弹窗
- 自动生成二维码

---

### 10. 原始文件页 (/:title)

**功能描述**：显示Gist的原始代码内容（纯文本）

**页面元素**：
- 无HTML界面
- 直接返回纯文本内容

**访问权限**：
- 公开Gist：所有人可访问
- 私有Gist：需登录

---

### 11. 错误页

**功能描述**：显示错误信息

**页面元素**：
- 错误消息

---

### 12. 私有Gist提示页

**功能描述**：提示用户需要登录才能访问私有Gist

**页面元素**：
- 提示消息
- 登录链接

---

- Logo + 应用名称（点击跳转到列表页）

**元素**：

## 数据模型

### Gist

| 字段 | 类型 | 说明 |
|------|------|------|
| ID | uint | 主键 |
| Title | string | 标题（唯一） |
| Filename | string | 文件名 |
| Content | string | 内容 |
| Visibility | int | 可见性（0=公开，1=私有） |
| CreatedAt | int64 | 创建时间（Unix时间戳） |
| UpdatedAt | int64 | 更新时间（Unix时间戳） |

### Revision

| 字段 | 类型 | 说明 |
|------|------|------|
| ID | uint | 主键 |
| GistID | uint | 关联的Gist ID |
| Version | int | 版本号 |
| Title | string | 标题 |
| Filename | string | 文件名 |
| Content | string | 内容 |
| Visibility | int | 可见性 |
| CreatedAt | int64 | 创建时间（Unix时间戳） |

### User

| 字段 | 类型 | 说明 |
|------|------|------|
| ID | uint | 主键 |
| Username | string | 用户名（固定为admin） |
| Password | string | 密码（加密存储） |
